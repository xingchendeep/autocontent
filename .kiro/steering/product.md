# Product: AutoContent Pro

A web tool for content creators to convert video scripts or links into platform-specific copy for 10 Chinese and international social platforms in one click.

## Core Value
Input a video script or URL → select target platforms → get ready-to-use copy for each platform in under 30 seconds.

## Target Users
- Individual creators managing multiple platforms (Douyin, Xiaohongshu, Bilibili, Weibo, WeChat, Twitter/X, LinkedIn, Kuaishou, Zhihu, Toutiao)
- Small content studios and MCN teams

## Supported Platforms
`douyin`, `xiaohongshu`, `bilibili`, `weibo`, `wechat`, `twitter`, `linkedin`, `kuaishou`, `zhihu`, `toutiao`

## Milestones
- **MVP**: Text input → platform selection → AI generation → copy result → local history (no login required)
- **v1.0**: User auth (Supabase), cloud history, usage stats, subscription payments (Lemon Squeezy)
- **v2.0**: Custom templates, batch processing, team collaboration, open API

## Key Constraints
- MVP requires no login to use core generation
- Generation success rate target: ≥ 95%
- Average generation time target: ≤ 30 seconds
- Primary language: Simplified Chinese
